# **App Name**: IC-SMART 2026 Portal

## Core Features:

- Conference Information Display: Present all essential conference details including event name, full title, organizer, dates, theme, about conference, thrust areas, registration fees, important dates, aims, publication information, contact details, and venue in a structured, accessible format.
- Dynamic Site Navigation: Implement a sticky header with smooth scrolling navigation, providing quick access to all key sections (Home, About, Call for Papers, Important Dates, Registration, Conference Aims, Contact, Venue) and individual pages.
- Live Conference Countdown: Display a real-time countdown timer prominently on the homepage, showing days, hours, minutes, and seconds remaining until the conference commencement.
- External Registration Link: Provide a clear 'Register Now' Call-to-Action (CTA) button, linking directly to the official registration form at https://forms.office.com/r/yjydBRn5Nx.
- Responsive Design for All Devices: Ensure the website layout and content adapt seamlessly across various screen sizes, offering an optimal viewing and interaction experience on desktops, tablets, and mobile devices.
- Conference Brochure Download: Include a dedicated button for users to easily download the conference brochure or other relevant documents.
- Interactive Visual Elements: Incorporate subtle animations for section reveals, light hover animations for interactive elements, and thematic tech-inspired background visuals (robotics, aerospace) to enhance user engagement.

## Style Guidelines:

- Primary interactive color: Teal (#1ecad3) to convey a modern, technological, and professional feel. This will be used for buttons, active navigation states, and key highlights, providing a vibrant contrast against the dark background.
- Background color: A deep, almost black, subtly teal-tinged dark navy (#1c2729), setting a professional and futuristic stage for the content while ensuring legibility of light text.
- Accent color: A bright, analogous cyan-green (#43e2a7) to draw attention to important calls-to-action or status indicators, creating a visually dynamic yet harmonious palette.
- Text accents: Crisp white (#FFFFFF) for primary text, ensuring maximum readability and a clean aesthetic against the dark background.
- Headline font: 'Space Grotesk' (sans-serif) for titles and headings, providing a contemporary, tech-inspired, and academic feel. This font supports the futuristic theme effectively.
- Body font: 'Inter' (sans-serif) for body text and descriptive content, chosen for its modern, clean, and highly readable characteristics suitable for detailed academic information.
- Utilize clean, vector-based icons that maintain a consistent, modern style. Integrate icons for social media links in the footer, adhering to the overall tech-inspired aesthetic.
- Implement a clean, spacious layout featuring modern card designs for various content sections to improve readability and visual hierarchy. Ensure responsive behavior across all screen sizes for an optimal mobile experience.
- Employ a sticky header for consistent navigation access and smooth scrolling throughout the site, enhancing user experience and site fluidity.
- Integrate subtle yet impactful animations, such as smooth transitions for section reveals and light hover effects on interactive elements, adding a modern and engaging touch without distracting from content.
